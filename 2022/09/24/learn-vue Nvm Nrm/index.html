Nvm

  • 管理node版本

  1. 下载nvm

    https://github.com/coreybutler/nvm-windows/releases

  2. nvm常用命令

    nvm list available 查看nodejs 官方的所有版本

    nvm list 查看安装的所有版本

    nvm install （版本号）下载对应的node版本号

    nvm use 切换node版本

Nrm

  • npm源管理器，可以在npm源之间进行切换

  1. 安装命令

    npm install -g nrm

  2. nrm常用命令

    nrm ls 查看源

    nrm use 切换源

    nrm add 名称 源 添加源

    nrm test npm 测试延迟
